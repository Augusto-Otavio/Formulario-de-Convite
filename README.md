<h1 align="center">🎉 Festinvite</h1>

<p align="center">
  <strong>Uma aplicação de convites digitais para eventos, feita com HTML e CSS.</strong>
</p>

<p align="center">
  Este projeto foi desenvolvido como parte do curso de Desenvolvedor Full Stack da <a href="https://www.rocketseat.com.br/" target="_blank">Rocketseat</a>, com o objetivo de criar uma interface moderna, acessível e intuitiva para criação de convites online, utilizando apenas <strong>HTML5</strong> e <strong>CSS3</strong>.
</p>

<h2>🚀 Funcionalidades</h2>

<h3>🗓️ Cadastro do Evento</h3>
<ul>
  <li>📝 Título do evento</li>
  <li>📅 Data e hora de início e término</li>
  <li>🧭 Tipo do evento: Presencial ou Online</li>
  <li>📍 Endereço físico ou link do evento</li>
  <li>🖊️ Descrição personalizada do evento</li>
</ul>

<h3>🎨 Personalização Visual</h3>
<ul>
  <li>🎨 Escolha da cor principal do convite</li>
  <li>💌 Seleção de temas (Aniversário, Formatura, Infantil, Casamento etc.)</li>
  <li>🌗 Alternância entre modo escuro e claro</li>
  <li>🖼️ Upload de imagem de capa</li>
</ul>

<h3>📇 Informações de Contato</h3>
<ul>
  <li>🙋 Nome completo (campo obrigatório)</li>
  <li>📧 E-mail e 📱 telefone</li>
  <li>🔔 Opção para receber atualizações por e-mail ou SMS</li>
  <li>✅ Confirmação dos Termos e Condições</li>
</ul>

<hr>

<h2>🛠️ Tecnologias Utilizadas</h2>

<h3>🔤 HTML5</h3>
<ul>
  <li>Estrutura semântica dos formulários</li>
  <li>Organização em seções acessíveis</li>
  <li>Uso correto de <code>label</code>, <code>fieldset</code> e <code>input</code></li>
</ul>

<h3>🎨 CSS3</h3>
<ul>
  <li>Estilização visual moderna com variáveis e pseudoclasses</li>
  <li>Layouts com <code>grid</code> e <code>flexbox</code></li>
  <li>Paleta de cores personalizável</li>
  <li>Estilo dark/light com alternância visual</li>
  <li>Validação visual de campos obrigatórios</li>
</ul>

<p><strong>⚠️ Este projeto ainda não é responsivo.</strong></p>

<hr>

<h2>📚 Aprendizados</h2>

<p>Durante o desenvolvimento do <strong>Festinvite</strong>, pude:</p>

<h3>✅ Trabalhar com diversos tipos de <code>&lt;input&gt;</code></h3>
<ul>
  <li><code>text</code>, <code>date</code>, <code>time</code>, <code>radio</code>, <code>file</code>, <code>checkbox</code></li>
  <li>Estruturação de formulários com <code>textarea</code> e campos obrigatórios</li>
</ul>

<h3>🎨 Explorar CSS com boas práticas</h3>
<ul>
  <li>Criação de componentes visuais modernos</li>
  <li>Validação de campos via pseudoclasses</li>
  <li>Estilização clara para formulários e feedbacks de erro</li>
  <li>Separação lógica das seções para melhor legibilidade</li>
</ul>
